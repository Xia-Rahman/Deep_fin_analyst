# Impact of New Labour Code on Indian Market
=============================================

## Executive Summary
--------------------

The new labour code, which was implemented in India in 2020, has had a significant impact on the country's market. The code has brought about changes in the way businesses operate, affecting both employees and employers. In this report, we will examine the impact of the new labour code on the Indian market, highlighting both the benefits and challenges that have arisen.

## Background
-------------

The new labour code was introduced to simplify and consolidate India's labour laws, which were previously complex and fragmented. The code has four main pillars:

* **Wages**: The code provides for a minimum wage guarantee and equal pay for equal work.
* **Industrial Relations**: The code aims to promote harmonious industrial relations by regulating the formation of trade unions and providing for dispute resolution mechanisms.
* **Occupational Safety, Health and Working Conditions**: The code sets standards for occupational safety, health, and working conditions, and provides for the protection of workers' rights.
* **Social Security**: The code provides for social security benefits, including pension, health insurance, and unemployment benefits.

## Impact on Businesses
---------------------

The new labour code has had a significant impact on businesses in India, both positive and negative. Some of the key effects include:

### Positive impacts

* **Simplified compliance**: The code has simplified the compliance process for businesses, reducing the administrative burden and costs associated with complying with multiple labour laws.
* **Increased transparency**: The code has increased transparency in the workplace, making it easier for employees to understand their rights and benefits.
* **Improved working conditions**: The code has set standards for occupational safety, health, and working conditions, improving the overall work environment for employees.

### Negative impacts

* **Increased costs**: The code has led to increased costs for businesses, particularly small and medium-sized enterprises (SMEs), which may struggle to absorb the costs of implementing the new regulations.
* **Labour unrest**: The code has led to labour unrest in some sectors, particularly in industries where workers are seeking better wages and working conditions.
* **Compliance challenges**: The code has presented challenges for businesses in terms of compliance, particularly in industries with complex supply chains.
